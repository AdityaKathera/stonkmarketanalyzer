## Mobile-First Improvements

### Touch Targets
```css
/* Minimum 44x44px for touch */

  min-height: 44px;
  min-width: 44px;
}
```

### Responsive Breakpoints
```css
/* Mobile first */
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
```
